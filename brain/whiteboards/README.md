# whiteboards/

Free-form excalidraw canvases not tied to a specific agent. For:

- Cross-domain brainstorming (body + finances + god decisions intersecting)
- Life-design maps, vision boards
- Project-level diagrams for avrystroeve.com itself
- Throwaway sketches and exploration

## Distinction from agent-co-located canvases

A canvas distilled from a specific source belongs INSIDE the relevant agent's `references/` folder, next to the source. Example: a whiteboard distilling the herbalist conversation lives at `body/herbalist/references/2026-05-15-herb-protocols.excalidraw`.

This folder is for canvases that DON'T cleanly belong to one agent.
