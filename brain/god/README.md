# god/ — spiritual / divine domain

Single-agent domain. The God agent is Avry's spiritual companion: drawing on the world's wisdom traditions (Yogananda / SRF, ancient mysticism, Hebrew letter mysticism, Christian contemplative, Vedic, Sufi, indigenous), Avry's personal practice, and his core conviction that God / Infinity / Allah / Dios / Aum is #1 and everything else is expression.

## Folder shape

- `SKILL.md` — agent definition (name, description, system prompt, when-to-invoke)
- `references/` — curated knowledge: practices, frameworks, distilled wisdom from sources, personal decisions about spiritual life
- `assets/` — templates the agent uses in output (prayer formats, meditation scripts, journaling prompts)
- `scripts/` — deterministic actions (if any emerge — none currently)

## When this agent is consulted

- Questions about spiritual practice, meditation, breathwork, prayer
- Wisdom-tradition lookups (Yogananda, Vedic, Sufi, Hebrew mysticism, Christian contemplative)
- Reflection on personal practice — what's working, what to deepen
- Discernment: "what does God / the divine seem to be saying about X?"
- Reading recommendations from canonical spiritual texts

## What this agent does NOT cover

- Body / health practice → see [body/](../body/)
- Money / runway / livelihood → see [finances/](../finances/)
- Project management for the website → see [PROJECT.md](../PROJECT.md)
