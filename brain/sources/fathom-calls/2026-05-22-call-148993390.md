  - [**Use the C.I.D.I. Framework:** Structure prompts with **C**ontext, **I**nstructions, **D**ata, and **I**nput. This yields more detailed, actionable plans than simple requests.](https://fathom.video/calls/684483527?tab=summary&timestamp=2350.0)

## Topics

### The Automation Challenge: Owner Statements

  - [**Goal:** Automate monthly owner statement generation, a manual process taking 6–10 days.](https://fathom.video/calls/684483527?tab=summary&timestamp=1140.0)
  - [**Problem:** The bottleneck is not the final report, but the manual data collection from disparate sources:](https://fathom.video/calls/684483527?tab=summary&timestamp=6861.0)
      - [Bank statements (\~1,500 transactions/month)](https://fathom.video/calls/684483527?tab=summary&timestamp=636.0)
      - [Reservation system reports](https://fathom.video/calls/684483527?tab=summary&timestamp=1210.0)
      - [Expense receipts](https://fathom.video/calls/684483527?tab=summary&timestamp=1182.0)
      - [Maintenance/cleaning time sheets](https://fathom.video/calls/684483527?tab=summary&timestamp=1226.0)
  - [**Root Cause:** Inconsistent data (e.g., multiple names for one property) causes AI inaccuracies.](https://fathom.video/calls/684483527?tab=summary&timestamp=5401.0)

### Solution 1: C.I.D.I. Prompting Framework

  - [A structured prompt framework (Context, Instructions, Data, Input) yields significantly better results than simple requests.](https://fathom.video/calls/684483527?tab=summary&timestamp=2350.0)
  - [**Example:**](https://fathom.video/calls/684483527?tab=summary&timestamp=2350.0)
      - [**Simple Prompt:** "I need a plan to process data for owner statements." → Vague response.](https://fathom.video/calls/684483527?tab=summary&timestamp=1853.0)
      - [**C.I.D.I. Prompt:** "You are an expert in real estate finance. I need a 100% accurate plan to create owner statements from bank data, reservation reports, and expense receipts. My current process is manual and time-consuming." → Detailed, multi-step plan.](https://fathom.video/calls/684483527?tab=summary&timestamp=2691.0)

### Solution 2: Claude's CoWork & Project Structure

  - [**CoWork vs. Chat:** CoWork links to a local computer folder, giving Claude direct access to files. This is more powerful than the web-based Chat.](https://fathom.video/calls/684483527?tab=summary&timestamp=4325.0)
  - [**Project Structure:** Create separate CoWork projects for distinct tasks (e.g., "Invoice Processing," "Owner Statements"). This isolates context and conserves tokens.](https://fathom.video/calls/684483527?tab=summary&timestamp=9300.0)
  - [**Connectors:** Link Claude to apps like Google Drive and Gmail.](https://fathom.video/calls/684483527?tab=summary&timestamp=5142.0)
      - [**Best Practice:** Connect only necessary apps to avoid token waste.](https://fathom.video/calls/684483527?tab=summary&timestamp=5286.0)
      - [**Example:** A prompt to find a `.md` file in Gmail failed because the email was routed through an internal server, not directly to the connected Gmail account.](https://fathom.video/calls/684483527?tab=summary&timestamp=5837.0)

### Solution 3: A Multi-Agent Automation Strategy

  - [**Concept:** Build a "team" of specialized AI agents for small, automatable tasks. An "orchestrator" agent can then manage them.](https://fathom.video/calls/684483527?tab=summary&timestamp=6305.0)
  - [**Example Use Case: Invoice Processing**](https://fathom.video/calls/684483527?tab=summary&timestamp=5880.0)
      - [**Trigger:** New email with an invoice.](https://fathom.video/calls/684483527?tab=summary&timestamp=6996.0)
      - [**Agent 1 (Validator):** Reads the PDF/XML to check the legal entity ID (`cédula jurídica`).](https://fathom.video/calls/684483527?tab=summary&timestamp=5940.0)
      - [**Agent 2 (Classifier):** If valid, downloads the PDF and files it in the correct property/month folder.](https://fathom.video/calls/684483527?tab=summary&timestamp=5907.0)
      - [**Agent 3 (Forwarder):** Sends the invoice to the property's accountant.](https://fathom.video/calls/684483527?tab=summary&timestamp=5959.0)
  - [**Skills & Plugins:** Use Claude's pre-built skills (e.g., `finance`) to give agents specialized knowledge, eliminating the need to provide that context in every prompt.](https://fathom.video/calls/684483527?tab=summary&timestamp=8040.0)
